# 🧑‍💻 NeetCode.io Progress Tracker

This repository documents my journey solving problems from **NeetCode.io** — a structured platform for coding interview preparation.  
I’m using this repo to keep track of solutions, notes, and approaches for different problem sets.
---

## Repository structure

Solutions are organized by topic folder, then problem ID. Each submission is stored as a separate file:

```
<topic-folder>/
  <problem-id>/
    submission-0.<ext>   ← first submission
    submission-1.<ext>   ← second submission
    ...
```


## 🎯 Goals

- Solve at least **one problem per day**  
- Build mastery across **arrays, strings, graphs, and dynamic programming**  
- Progress from **NeetCode 150 → NeetCode 250**  
- Document solutions with clear explanations and optimized code
